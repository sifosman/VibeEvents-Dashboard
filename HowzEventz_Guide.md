# HowzEventz Application Guide

## Overview

HowzEventz is a comprehensive event planning platform designed to connect event hosts with service providers. The application provides tools for discovering vendors, planning events, managing bookings, and tracking budgets.

## Key Features Implemented

1. **Enhanced Navigation**
   - Dedicated sections for "For Vendors" and "For Service Providers"
   - Appropriate subcategories in both desktop and mobile views

2. **Mobile Bottom Navigation**
   - Navigate (Home)
   - Search by Category
   - My Shortlist (Liked Items)
   - My Planning
   - My Account

3. **My Account Page**
   - Profile management
   - Password settings
   - Application settings
   - FAQs
   - Ratings & Reviews
   - Log Out functionality

4. **My Planning Page**
   - Events Diary
   - Quotes management
   - Booking Confirmations
   - Budget Tracker

## My Planning Page

The My Planning page serves as a central hub for all event planning activities:

### Events Diary
- Track your event timeline and key dates
- Add new events and schedule important milestones
- Set reminders for critical deadlines

### Quotes
- Request and compare quotes from different vendors
- Track quote statuses and response times
- Download quote templates for various vendor categories

### Booking Confirmations
- Store and manage all your vendor contracts
- Track payment schedules and upcoming payments
- Use the contract checklist to ensure all important details are covered

### Budget Tracker
- Set your total event budget and category-specific budgets
- Track expenses against your budget
- Visualize your spending distribution
- Access budget templates for different event types

## How to Use

1. **Navigating the App**
   - Use the bottom navigation on mobile devices to access key sections
   - Use the top navigation on desktop to access detailed categories

2. **Planning an Event**
   - Start with the My Planning page to access all planning tools
   - Create an event in the Events Diary
   - Request quotes from vendors
   - Track confirmed bookings
   - Monitor your budget

3. **Managing Your Account**
   - Access the My Account page to update your profile
   - Manage password and security settings
   - Customize your application preferences
